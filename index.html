Assemble

Introduction

Welcome to Assemble!

Designed at the outset for role-playing system designers, Assemble is a content-authorship tool which allows for RPG designers to plan and write Powered by the Apocalypse (PbtA) systems.

Assemble is an open-source web application written in JavaScript and Built with Blockly. View Assemble on Github!

Chrome is recommended to most effectively display Assemble.

The game is set in a post-apocalyptic wasteland. Scarcity The motivations of characters are driven by want due to a lack of resources. Technology Most technology in the setting is based on the use of scrap metal, or jury-rigging artifacts from the world before. The Unknown The apocalypse makes both the past and present uncertain, reducing the players' overall sense of security.
